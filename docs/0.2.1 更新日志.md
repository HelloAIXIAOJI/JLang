# JsonLang 0.2.1 更新日志

## 新特性
- **递归函数调用支持**：完整支持递归函数调用，使复杂算法的实现成为可能
- **函数局部变量作用域**：优化函数调用过程中的变量管理，避免变量污染
- **嵌套数据访问**：支持多层嵌套的数据结构访问，如 `nested_data.level1.level2.value`

## 优化
- **上下文状态管理**：执行函数前保存上下文状态，执行后恢复变量环境
- **返回值机制**：支持多种返回值变量名，增强函数间数据传递的灵活性
- **函数参数处理**：改进函数参数解析和变量绑定机制

- 修复嵌套函数调用时返回值丢失的问题
